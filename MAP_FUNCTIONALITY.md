## Interactive Map Functionality

This document outlines the features, behaviors, and user interactions for the interactive map functionality of the campaign manager app. The map serves as a core tool for data visualization and strategic management in the Star Wars: The Old Republic roleplaying community.

### Overview
The interactive map provides the following key capabilities:
- Displaying strategic, geographic, and political information.
- Tracking story events, assets, and in-universe elements.
- Enabling user-specific views for players and GMs, with gated access.
- Supporting in-app dataset updates through marker creation, management, and deletion.
